Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents and manipulates HTML and XML documents as a structured tree of objects.

How do you select an element form the DOM ?

You can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName().

what is event delgation in the context of the DOM ,and Why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for multiple child elements, improving performance and simplifying code.

How do you mainpulate attributes and style using the DOM?

You can manipulate attributes using element.setAttribute() and element.getAttribute(), and change styles with element.style.propertyName in the DOM.
